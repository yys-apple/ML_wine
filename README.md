<H1>A mini Machine Learning project</H1>
Use wine dataset.

Required packages:
* scikit-learn
* pandas

In Windows, you can install the required packages using the following commands:
<B>pip install scikit-learn pandas</B>

In Ubuntu:
* sudo apt install python3-venv
* python3 -m venv .venv
* source .venv/bin/activate
* pip install scikit-learn pandas
